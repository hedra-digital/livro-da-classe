## Setup Guide (Ubuntu)

### Install Texlive

    sudo apt-get texlive-full

### Install ImageMagick

    sudo apt-get install libmagickwand-dev libmagickcore-dev imagemagick

### Symlink LaTeX template directory

    sudo ln -s /etc/texmf/tex/generic/ ~/apps/livrodaclasse/current/templates
